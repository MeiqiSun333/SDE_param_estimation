# Raw Data

Raw longitudinal emotion survey data is not included in this repository due to privacy and ethical considerations.

## Expected Format

If using your own data, ensure the CSV file contains:
- `UUID`: Participant identifier
- `Date_Local`: Date (DD/MM/YYYY)
- `Time_Local`: Time (HH:MM:SS)
- `SAD_ES`, `RUM_ES`, `DIST_ES`, `EMOCOPE_ES`: Emotion ratings

Place your raw data file here and run the preprocessing pipeline.
